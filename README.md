This project focuses on developing digital literacy skills through practical tasks such as creating digital content, building an online portfolio, using digital platforms, practicing email etiquette, and understanding cybercrime awareness.

The objective of this project is to enhance knowledge of digital tools and promote safe and responsible use of technology.
